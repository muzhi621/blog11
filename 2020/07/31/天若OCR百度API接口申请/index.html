天若OCR百度API接口申请
发表于2020-07-31|更新于2020-07-31|技术
|阅读量:

天若大佬的开源项目里面，软件本身提供了一个百度接口的更换，就在软件的设置里面，而且天若OCR在百度接口的调用上包含了很多功能，其中就有表格识别。所以，我们只用申请一个百度API接口，就可以用啦，免费次数足够个人使用，每天50次免费。

先给大家看一下调用识别测试


下面是百度API接口申请教程：
一、开始申请
首先需要登录平台：https://login.bce.baidu.com/
建议使用百度账号登录，云账号不能申请内测功能（如公式识别）
登陆之后跳转到表格文字识别申请入口：https://ai.baidu.com/tech/ocr_others/table

第一次使用，根据提示进行下一步。
二、应用创建
进入文字识别管理页面进行应用创建

三、获取密钥

四、软件设置
这里提供两种方法，第一种可能会失效，第二种一劳永逸（推荐）！
（1）进入天若OCR设置里面，点击密钥，将上面划线部分获取到的API Key和Secret Key分别填入百度接口的账号和密码里，填完之后退出软件重新打开看是否填写成功。

（2）不从软件设置里面修改，找到天若OCR软件所在目录，打开所在目录下的“Data”文件夹—>打开config（这是设置的配置参数文件），将上面划线部分获取到的API Key和Secret Key分别填入[密钥_百度]的id和key里，退出并保存文件，右键属性，把文件修改成“”只读“”，应用。打开检查一下是否保存成功，成功的话打开软件，进入设置，此时的账号密码是刚刚自己填入的。
提醒：不要点击设置里面的“”恢复默认“”按钮，点击的话还是会恢复成原来的，因为默认账户在软件代码里面保存着，楼主也没找到在什么位置！！！
当然如果不小心点击的话，退出重新打开软件就行了，因为config已经被设为只读了。

填写成功后进入软件界面右键换接口为表格—>百度，找一个表格图片调用测试一下，成功的话就ok啦。

这里提供一张表格供大家测试，楼主测试5次左右都识别了。楼主发现的确存在不能识别的图片，可能是图片分辨率不够，识别未成功就会返回密钥次数已用完，如出现这种情况，先检查自己当日50次是否用完，没用完换一张图片试一试，比如下面这张。

经测试，换完自己的密钥之后，关于百度接口的功能都能使用啦。包括文字识别，把接口换成百度—>中英就行了。再也不用担心软件出现“*\该区域未发现文本***”了！！！
楼主不是技术大佬，只能在天若OCR原有基础上把不能用的功能换个接口使用，能力有限。其实这款软件真的特别好，而且这个软件开源了，如果有技术大佬感兴趣的话希望能继续维护一下。
另外，其实这个软件天若幽心大佬一直在做，功能增加了很多，特别是接口可以直接更换，有需要的朋友多关注，软件地址：https://tianruoocr.cn/
